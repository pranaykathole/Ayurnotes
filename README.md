# AyurNotes – BAMS Notes Website

AyurNotes is a simple website where students can view and download BAMS study notes.  
Admin can upload PDFs through a secure login panel.  
The site uses Firebase Firestore + Firebase Storage + Firebase Auth.

---

## ✅ Features
- View uploaded notes
- Admin login (Email/Password)
- Upload PDF notes
- Automatic listing of notes
- Mobile-friendly UI
- Works on GitHub Pages, Vercel, Netlify, or any static hosting
